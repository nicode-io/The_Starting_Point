# Description

TypeScript is a strict syntactical superset of JavaScript and adds optional static typing to the language. TypeScript is designed for the development of large applications and transcompiles to JavaScript. 

As TypeScript is a superset of JavaScript, existing JavaScript programs are also valid TypeScript programs.

Discover TypeScript and the advantage of data typing in JavaScript. Avoid a lot of errors and make code more reliable through TypeScript

#   Tips

*   TS can't add features that don't exist on JS. It can just simplify, beautify syntax before compiling. 
*   Don't give a type to a variable which you assign a value directly, only use type when not declaring a value.
*   Don't get confuse between js object (with ,) and data type object (with ;)
